# 케이스원 (Case One)

흩어진 프로젝트 재료(정량·정성·협업·결과물)를 근거 있는 케이스 스터디로 엮어주는 프로덕트 디자이너 도구.

## 구성

| 파일 | 설명 |
|---|---|
| `index.html` | 진입 페이지 — React 18 UMD(CDN) 로드 후 앱 실행 |
| `app.js` | 사전 컴파일된 앱 전체 (이미지 임베드 포함, 빌드 불필요) |

빌드 과정이 필요 없는 **정적 사이트**입니다.

## 배포 방법 (GitHub → Vercel)

1. **GitHub 저장소 만들기**
   - github.com에서 새 저장소 생성 (예: `case-one`)
   - 이 폴더의 `index.html`, `app.js`, `README.md`를 업로드
     (웹에서 "Add file → Upload files"로 끌어다 놓으면 됩니다)

2. **Vercel 연결**
   - vercel.com → "Add New… → Project" → 방금 만든 저장소 Import
   - Framework Preset: **Other**
   - Build Command / Output Directory: **비워두기** (기본값 그대로)
   - Deploy 클릭

3. 완료되면 `프로젝트명.vercel.app` 주소가 발급됩니다.

## 알아둘 것

- **데모 모드가 기본 시연 경로입니다.** 재료 입력 화면의 "데모 재료로 채우기"를 누르면
  실시간 주문 추적 사례의 재료가 순차적으로 채워지고, 전체 파이프라인
  (병렬 수집 → 기준선 판별 → 5단 서사)이 예시 결과로 재생됩니다.
- **실제 AI 생성**(직접 재료 입력 후 케이스 생성)은 Claude API 인증이 필요해서
  이 정적 배포에서는 동작하지 않습니다. 데모 모드로 시연해 주세요.
- 폰트(Google Sans Flex, Pretendard)는 CDN에서 로드됩니다.

## 사용 흐름

히어로 → 데모 시작 → 재료 4스텝(정량·정성·협업·결과물, 전부 선택 사항)
→ 케이스 생성 → 다크 생성 화면(병렬 수집·기준선 판별·5단 배치)
→ 케이스 스터디 (①기준선 → ②문제 → ③결정 → ④변화 → ⑤성과)

완성된 케이스는 우측 하단 "복사"로 이미지 포함 복사(노션 붙여넣기 지원)됩니다.
